# Mermaid（流程图绘制）



### 一、流程图

#### 1.横向流程图

```mermaid
%% LR表示从左到右布局，TB表示从上到下布局
graph LR
A([椭圆形]) -->B(圆角)
    B --> C{菱形}
    C ---|线条无箭头| D[方形]
    C ===|粗线条无箭头| E[(圆柱形)]
    C ==>|粗线条箭头| F((圆形))
    C -->|粗线条箭头| G[[柱形]]
    C -.虚线条箭头.->H>箭头形状]
    C -.徐线条无箭头.-I[I]
```



#### 2. 竖向流程图


```mermaid
graph TD
A[方形] --> B(圆角)
    B --> C{条件a}
    C --> |a=1| D[结果1]
    C --> |a=2| E[结果2]
```
