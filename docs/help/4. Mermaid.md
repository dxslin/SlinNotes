# Mermaid（流程图绘制）

### 一、流程图



```mermaid
graph LR
A[方形] -->B(圆角)
    B --> C{条件a}
    C ---|线条无箭头| D[结果1]
    C ===|粗线条无箭头| E[结果2]
    C ==>|粗线条箭头| F[结果3]
    C -->|粗线条箭头| G[结果3]
    
    Title[横向流程图]
```


```mermaid
graph TD
A[方形] --> B(圆角)
    B --> C{条件a}
    C --> |a=1| D[结果1]
    C --> |a=2| E[结果2]
    F[竖向流程图]
```



### 二、时序图

```mermaid
sequenceDiagram
对象A->>对象B: 对象B你好吗?（请求）
Note right of 对象B: 对象B的描述
Note left of 对象A: 对象A的描述(提示)
对象B-->对象A: 我很好(响应)
对象A->对象B: 你真的好吗？
```


```mermaid
sequenceDiagram
对象A->对象B: 对象B你好吗?（请求）
Note right of 对象B: 对象B的描述
Note left of 对象A: 对象A的描述(提示)
对象B-->对象A: 我很好(响应)
对象B->小三: 你好吗
小三-->>对象A: 对象B找我了
对象A->对象B: 你真的好吗？
Note over 小三,对象B: 我们是朋友
participant C
Note right of C: 没人陪我玩
```

gantt

title 软件开发甘特图

    需求                    :done,    des1, 2014-01-06,2014-01-08
