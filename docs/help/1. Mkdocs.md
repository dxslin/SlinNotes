#  Mkdocs

完整文档请访问 [mkdocs.org](https://www.mkdocs.org).

## 命令

* `mkdocs new [dir-name]` - 创建一个新的工程。
* `mkdocs serve` - 启动实时预览服务。
* `mkdocs build` - 编译文档网站，会应用配置文件`mkdocs.yml`中的site_url。
* `mkdocs -h` - 帮助和退出

## 项目结构

    mkdocs.yml    # 配置文件，配置网站名称和域名。
    docs/
        index.md  # 文档首页
        ...       # 其他页面，如果是文件夹会形成下拉菜单，如果是文件，则直接引用访问
